# South African Human Trafficking Victim Data Collection Template - Version 1.0

## Purpose
This template provides a standardized framework for collecting data on human trafficking victims in South Africa. Consistent data collection across all stakeholders (law enforcement, social services, healthcare providers, NGOs) is essential for accurate analysis, effective resource allocation, and informed policy-making.

## Instructions for Use
1. **Confidentiality**: All data must be collected and stored in accordance with South African data protection laws (POPIA). Personally identifiable information (PII) must be anonymized.
2. **Completeness**: Fill out all applicable sections. If a field is not applicable, mark as "N/A".
3. **Consistency**: Use the specified codes and categories to ensure data uniformity.
